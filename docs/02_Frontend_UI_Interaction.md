## 3. 全局交互规则

### 3.1 自动保存

以下内容需要自动保存：

1. 项目配置
2. 输入文本
3. 分集大纲修改
4. 剧本修改
5. 角色设定
6. 声音绑定

交互要求：

- 输入后 1-2 秒防抖保存。
- 保存中显示“保存中”。
- 保存成功显示“已保存”。
- 保存失败显示“保存失败，点击重试”。

### 3.2 长任务状态

Agent 生成、音频生成、混音属于长任务。

统一状态：

1. pending：等待中
2. running：生成中
3. succeeded：成功
4. failed：失败
5. cancelled：已取消

UI 要求：

- 显示任务进度状态。
- 支持刷新页面后继续看到状态。
- 支持失败后重试。
- 支持查看错误信息。

### 3.3 锁定机制

用户可以锁定：

1. 单集大纲
2. 剧本块
3. 角色设定
4. 角色声音
5. 最终音频版本

UI 表现：

- 锁定后显示锁图标。
- 重新生成时明确提示“已锁定内容不会被覆盖”。
- 用户可手动解除锁定。

### 3.4 人工确认节点

以下节点必须人工确认：

1. 分集大纲确认
2. 剧本审核确认
3. 角色声音确认
4. 最终音频确认

## 6. 新建项目页

### 6.1 路由

`/projects/new`

### 6.2 页面目标

通过向导完成项目创建。

### 6.3 向导步骤

#### Step 1：基础信息

字段：

1. 项目名称，必填。
2. 项目简介，可选。
3. 项目类型，必选。

项目类型选项：

- 原著改编
- 单篇故事改编
- 主题原创
- 知识科普

#### Step 2：内容来源

选项：

1. 粘贴文本
2. 上传 TXT / Markdown
3. 输入故事主题

粘贴文本字段：

- 文本输入框
- 字数统计
- 清洗文本按钮

上传文件字段：

- 文件上传区域
- 支持格式说明
- 文件名
- 文件大小
- 上传状态

输入主题字段：

- 主题输入框
- 补充说明输入框

#### Step 3：目标配置

字段：

1. 目标年龄：3-5 岁 / 5-8 岁 / 8-12 岁
2. 计划集数：数字输入，默认 3
3. 单集时长：3 / 5 / 8 / 10 / 12 分钟，也可自定义
4. 音频形式：单人讲述 / 旁白 + 多角色对话 / 儿童广播剧 / 睡前故事
5. 原著忠实度：低 / 中 / 高

#### Step 4：风格选择

风格卡片：

1. 经典儿童广播故事风
2. 孙敬修式儿童故事感
3. 睡前陪伴风
4. 冒险喜剧风
5. 国学启蒙风
6. 温柔治愈风

每张卡片包含：

- 风格名称
- 一句话说明
- 适合年龄段
- 适合故事类型

#### Step 5：确认创建

展示项目配置摘要。

按钮：

- 上一步
- 创建项目并解析素材

### 6.4 创建成功

成功后跳转：

`/projects/:projectId/source`

4. 通过后将大纲状态改为 `outline_approved`。
5. 显示“生成剧本”按钮。

1. 块类型：旁白 / 台词 / 音效 / BGM / 停顿 / 转场
2. 角色
3. 正文
4. 情绪
5. 语速
6. 停顿
7. 声音
8. 音频生成状态
9. 审核状态
10. 锁定状态

### 10.6 剧本块操作

用户可以：

1. 编辑文本
2. 修改角色
3. 修改块类型
4. 修改情绪
5. 修改语速
6. 添加停顿
7. 上移 / 下移
8. 新增块
9. 删除块
10. 锁定块
11. 局部重写
12. 生成此句音频
13. 试听此句音频

### 10.7 局部重写弹窗

用户选中剧本块后，点击“重写”。

重写方向：

1. 更适合 3-5 岁
2. 更活泼
3. 更温柔
4. 更像广播故事
5. 缩短
6. 扩写
7. 降低刺激感
8. 增加互动提问
9. 增加幽默
10. 保留意思但换一种说法

确认后调用局部重写接口。

展示：

1. 角色名称
2. 性格
3. 声音建议

展示：

1. 风险列表
2. 风险等级
3. 风险位置
4. 修改建议
5. 一键改写按钮

### 10.10 剧本审核确认

当所有高风险问题处理后，用户可以点击“确认剧本”。

确认后：

1. 剧本状态改为 `script_approved`。
2. 已确认剧本进入声音选择流程。
3. 后续重新生成需提示会覆盖未锁定内容。

### 11.4 角色列表字段

1. 声音名称

当所有必需角色都绑定声音后，用户可以点击“确认声音并进入音频生成”。

## 12. 音频生成页

### 12.1 路由

`/projects/:projectId/audio`

### 12.2 页面目标

### 12.3 页面结构

1. 集数列表
2. 生成状态
3. 音频参数
4. 生成按钮
5. 错误信息
6. 任务日志

### 12.4 音频参数

1. 语速：慢 / 标准 / 快
2. 情绪强度：低 / 中 / 高
3. BGM：开启 / 关闭
4. 音效：开启 / 关闭
5. BGM 音量：0-100
6. 输出格式：MP3，后续支持 WAV

### 12.5 生成操作

支持：

1. 生成当前集
2. 生成全部已审核集
3. 重新生成失败片段
4. 重新混音

### 12.6 生成中状态

显示：

1. 当前任务名称
2. 当前集数
3. 已完成块数 / 总块数
4. 当前阶段：TTS 生成 / 音效处理 / BGM 处理 / 混音 / 导出
5. 失败原因
