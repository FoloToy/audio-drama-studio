# 儿童音频短剧创作 Agent 平台文档包

本目录用于交给 Codex / 研发团队继续开发使用。

## 文件列表

1. `01_PRD.md`  
   产品需求文档，说明产品定位、目标用户、MVP 范围、核心流程、数据结构和验收标准。

2. `02_Frontend_UI_Interaction.md`  
   前端页面与交互说明，覆盖页面结构、用户路径、组件、状态、表单字段、编辑器交互、试听与导出流程。

3. `03_Agent_Architecture_API.md`  
   Agent 技术架构与接口定义，覆盖 Agent 编排、服务模块、核心 API、数据模型、任务队列、生成流程、审核机制和错误处理。

## 推荐开发顺序

1. 先根据 `01_PRD.md` 确认 MVP 范围。
2. 再根据 `02_Frontend_UI_Interaction.md` 搭建前端页面和交互骨架。
3. 同步根据 `03_Agent_Architecture_API.md` 设计后端服务、Agent 编排和接口。
4. 第一阶段优先打通：项目创建 → 文本输入 → 分集大纲 → 剧本生成 → 人工审核 → 声音绑定 → 音频生成 → MP3 导出。

## MVP 优先级

P0 必须完成：

- 项目管理
- 内容输入
- 目标年龄、集数、风格配置
- 素材解析
- 分集大纲生成与编辑
- 剧本生成与编辑
- 儿童安全审核
- 角色识别
- 声音绑定
- 单集音频生成
- 试听与导出

P1 可后续增强：

- PDF / EPUB / Word 上传
- 批量生成全项目音频
- 音效与 BGM 自动匹配
- 分轨导出
- 多人协作
- IP 世界观管理
